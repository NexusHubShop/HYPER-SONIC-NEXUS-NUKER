node -v
npm -v

If both commands return version numbers, the installation was successful.

3) Navigate to Your Nuker Folder

Go to the folder where your bot files are stored.

If the folder is directly in internal storage:

cd shared

If the folder is inside Downloads:

cd storage
cd shared
cd Download
cd folder-name

Replace "BOT-FOLDER" with your actual folder name.

4) Install Required Packages

Install the required dependencies:

Finally, start the bot using:
